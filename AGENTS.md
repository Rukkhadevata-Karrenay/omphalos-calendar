# AGENTS.md

## 项目目标

这是一个以《崩坏：星穹铁道》翁法罗斯一年历为主题的本地网页日历软件。

项目重点不是普通日历，而是将现实日期、现实时间和翁法罗斯历法主题结合，形成沉浸式时间管理工具。

本项目为非官方粉丝作品，不隶属于 HoYoverse / miHoYo，不用于商业用途。游戏相关名称与素材版权归原权利方所有。

## 技术栈

- React
- Vite
- TypeScript
- CSS
- localStorage

## 核心规则

1. 每次修改后必须运行 `npm run build`。
2. 不要随意引入复杂依赖。
3. 不要把所有逻辑堆在 `App.tsx`。
4. 翁法罗斯月份数据放在 `src/data/omphalosMonths.ts`。
5. 翁法罗斯时间数据放在 `src/data/omphalosTime.ts`。
6. 时间换算逻辑放在 `src/utils/omphalosTime.ts`。
7. 日程数据必须兼容 `localStorage`。
8. 修改日程结构时必须考虑旧数据迁移。
9. UI 风格保持深色、神话感、玻璃拟态、金色细线装饰。
10. 本项目是非官方粉丝作品，不得写成官方项目。
11. 不直接引用 `IMG_*.jpg` 这类无意义文件名，业务代码只引用语义文件名。
12. `public/assets/omphalos/original/` 只作原图备份，不作为业务 UI 入口。

## 翁法罗斯一天五时段

1. 门扉时：00:00:00 - 04:48:00
2. 明晰时 / 上升时：04:48:01 - 09:36:00
3. 践行时 / 下降时：09:36:01 - 14:24:00
4. 离愁时：14:24:01 - 19:12:00
5. 幕匿时：19:12:01 - 24:00:00

一天分为 5 个时段，每个时段 4 小时 48 分钟。每个时段分为 5 刻，每刻 57 分 36 秒。

## 开发优先级

第一优先级：

- 页面能运行
- 日程能添加、编辑、删除
- 数据能保存
- 翁法罗斯时间换算准确

第二优先级：

- 视觉风格
- 月份主题图
- 动态时段展示

第三优先级：

- 搜索
- 统计
- 导入导出
- 发布展示
